# ASTRA – Autism Spectrum Therapeutic & Resource Assistant

## Overview
ASTRA is an AI-powered platform designed to assist in early autism screening, therapy support, and progress tracking for children with Autism Spectrum Disorder (ASD). The system uses Machine Learning and Computer Vision to analyze behavioral patterns and provide interactive therapy activities.

The goal of ASTRA is to support parents, caregivers, and educators by providing early risk detection, engaging therapy exercises, and progress monitoring tools.

---

## Features
- Autism Risk Screening using Machine Learning
- Computer Vision-based Activity Tracking
- Interactive Therapy Games for Children
- Progress Tracking Dashboard
- Personalized Activity Recommendations
- Parent-Friendly Reports
- Real-time Interaction Monitoring

---

## Key Modules
### 1. Intelligent Screening
- Uses ML models to analyze questionnaire data
- Predicts autism risk level
- Provides early screening assistance

### 2. Computer Vision Therapy
- Face tracking
- Eye contact detection
- Hand gesture tracking
- Interactive activities like:
  - Magic Canvas (Air Drawing)
  - Magic Drum
  - Emoji Matching Game

### 3. Progress Dashboard
- Weekly activity report
- Performance tracking
- Completed vs Skipped activities
- Improvement monitoring

### 4. Recommendation System
- Suggests therapy activities
- Suggests lifestyle improvements
- Generates parent reports

## System Workflow
1. User fills screening questionnaire
2. ML model predicts autism risk
3. Child performs interactive therapy activities
4. Computer Vision tracks activity performance
5. System stores performance data
6. Dashboard shows progress reports
7. System generates recommendations
